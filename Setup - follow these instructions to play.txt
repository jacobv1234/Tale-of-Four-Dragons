NOTE - this game requires Python to play. This can be downloaded from https://www.python.org/downloads/

To create a shortcut:
1) right-click the program and select 'Create Shortcut' - name it 'The Tale of Four Dragons'
2) right-click the shorcut and go to properties
3) in 'change icon' navigate to the game's folder and select 'icon.ico' to use.

You can have multiple save files by:
1) create a new folder in the same folder as the program - name it whatever
2) within the folder create 5 text files called weapons, spells, inventory, flags and other (all lowercase)
3) the save file that will be used is the folder called SaveData - make sure no other folders share this name
